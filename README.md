# Addiction Uncovered

**An Educational Web App on Substance Abuse Awareness**

## Overview
Addiction Uncovered is an interactive educational app designed to teach users about substance abuse, prevention, and healthy coping strategies. The goal is to engage users without trivializing the topic.

## Features

### Pinpoint
- Daily word-association puzzle
- Guess the hidden category using clue words
- Themed around addiction awareness

### Jeopardy Challenge
- Classic quiz board with multiple categories
- Supports teams, scoring, and stealing

### Memory Match
- Card-matching game linking concepts
- Visual reinforcement of key terms

### Quick Fire Quiz
- Rapid true/false questions
- Instant feedback and score tracking

## Technology Stack
- HTML, Tailwind CSS, JavaScript
- Fully client-side, mobile-first design
- No server or database required

## Installation
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Navigate to any game mode from the homepage
